# BigQuery データ分析 練習プロジェクト

## 概要
BigQuery の公開データセットを使って、データ分析の基礎を学ぶプロジェクトです。

## 学べること
- **SQL**: データの取得・絞り込み・集計
- **可視化**: 棒グラフ、折れ線グラフ、円グラフ、ヒートマップ
- **統計分析**: 基本統計量、分布、相関分析

## セットアップ
このプロジェクトはセットアップ済みです。

## 使い方
1. Cursor で `bigquery_analysis.ipynb` を開く
2. カーネルに「BigQuery Analysis」を選択
3. 上から順にセルを実行（Shift + Enter）

## 使用データ
- `bigquery-public-data.austin_bikeshare.bikeshare_trips`（自転車シェアリングデータ）
